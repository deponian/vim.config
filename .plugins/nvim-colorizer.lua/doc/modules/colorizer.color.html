hsl_to_rgb (h, s, l)
Converts an HSL color value to RGB. Accepts hue, saturation, and lightness values, each within the range [0, 1], and converts them to an RGB color representation with values scaled to [0, 255].

Parameters:

  • h number: Hue, in the range [0, 1].
  • s number: Saturation, in the range [0, 1].
  • l number: Lightness, in the range [0, 1].

Returns:

  1. number|nil,number|nil,number|nil: Returns red, green, and blue values scaled to [0, 255], or nil if any input value is out of range.
  2. number|nil,number|nil,number|nil
hue_to_rgb (p, q, t)
Converts an HSL component to RGB, used within `hsl_to_rgb`. Source: https://gist.github.com/mjackson/5311256 This function computes one component of the RGB value by adjusting the color based on intermediate values `p`, `q`, and `t`.

Parameters:

  • p number: A helper variable representing part of the lightness scale.
  • q number: Another helper variable based on saturation and lightness.
  • t number: Adjusted hue component to be converted to RGB.

Returns:

    number: The RGB component value, in the range [0, 1].
